{# 1. Présentation #}

1. Présentation de la plateforme

KoboToolBox Dashboard est une application web qui se connecte à un serveur KoboToolBox et affiche les données de soumission de formulaires sous forme de tableaux de bord interactifs.

La plateforme est organisée en groupes : chaque groupe dispose d'un ensemble de formulaires accessibles et d'une liste de membres. Un utilisateur ne voit que les formulaires appartenant à ses groupes.

RôleAccès
Utilisateur Consulte les dashboards de ses groupes, utilise les filtres et exports.
Admin de groupe Tout ce qu'un utilisateur peut faire + gérer les membres de son groupe + charger des modules de dashboard.
Administrateur principal Accès complet : configuration du serveur, gestion de tous les formulaires, groupes et utilisateurs.
{# 2. Accès et connexion #}

2. Accès et connexion

Demander l'accès

Si vous n'avez pas encore de compte, cliquez sur « Créer un compte » sur la page de connexion. Renseignez votre nom complet et votre adresse email, puis choisissez un mot de passe. Votre demande sera transmise à un administrateur pour validation. Vous recevrez un message de confirmation une fois votre compte activé.

Se connecter

Saisissez votre adresse email et votre mot de passe sur la page de connexion (/accounts/login/). La session reste ouverte jusqu'à ce que vous cliquiez sur « Déconnexion » en haut à droite.

Mot de passe oublié

Contactez votre administrateur de groupe ou l'administrateur principal. Celui-ci peut générer un lien de réinitialisation de mot de passe et vous le transmettre directement.

{# 3. Dashboards #}

3. Consulter les dashboards

La page d'accueil (« Formulaires ») liste tous les dashboards auxquels vous avez accès. Cliquez sur « Visionner le dashboard » pour ouvrir un formulaire.

Deux types de vues existent selon le formulaire :

  • Dashboard indicateurs — affiche des cartes de synthèse (bénéficiaires totaux, rapports soumis), des graphiques par pays, par période et par résultat, une carte de désagrégation (âge/sexe, handicap, statut) et un tableau de données détaillé.
  • Matrice de couverture — affiche un tableau croisé activités × pays indiquant le nombre de soumissions pour chaque combinaison. Les cellules cliquables donnent accès à la liste des soumissions correspondantes.

Le bouton Rafraîchir (sur la page du dashboard) force le rechargement des données depuis le serveur KoboToolBox, en ignorant le cache.

{# 4. Filtres #}

4. Utiliser les filtres

Les dashboards proposent une barre de filtres en haut de page. Les filtres disponibles varient selon le type de formulaire :

  • Pays — restreint l'affichage à un pays ou une zone géographique.
  • Année / Trimestre — filtre par période de rapport.
  • Résultat — affiche uniquement les indicateurs d'un résultat donné.
  • Responsable — filtre par personne ou entité responsable de l'activité (matrice de couverture).

Les filtres sont cumulables. Pour les réinitialiser, cliquez sur « Réinitialiser » ou supprimez les paramètres de l'URL.

{# 5. Exports #}

5. Exporter les données

Deux formats d'export sont disponibles depuis la page du dashboard :

  • CSV — fichier texte séparé par des virgules, compatible avec la plupart des tableurs. Encodage UTF-8 avec BOM pour une compatibilité optimale avec Excel.
  • XLSX — fichier Excel natif avec en-têtes formatées et colonnes ajustées automatiquement.

L'export porte sur l'ensemble des données du formulaire, pas uniquement sur les lignes filtrées à l'écran.

{# 6. Admin de groupe #}

6. Administrateurs de groupe

Si vous êtes administrateur d'un groupe, l'icône apparaît dans la barre de navigation. Elle donne accès à la page « Mon groupe ».

Gérer les membres
  • Ajouter un membre — sélectionnez un utilisateur actif dans la liste déroulante et cliquez sur « Ajouter ». Seuls les utilisateurs déjà créés et activés par l'administrateur principal apparaissent dans la liste.
  • Désactiver un compte — l'utilisateur ne peut plus se connecter mais son compte est conservé. Cliquez sur .
  • Supprimer un compte — suppression définitive. Cliquez sur et confirmez.
  • Réinitialiser le mot de passe — cliquez sur pour générer un lien de réinitialisation à usage unique (valable 3 jours). Copiez ce lien et transmettez-le à l'utilisateur par email ou messagerie.
Gérer les modules de dashboard

Chaque formulaire peut avoir un fichier de module Python (.py) qui définit comment les soumissions sont analysées et affichées. En tant qu'admin de groupe, vous pouvez :

  • Télécharger le module actuel — cliquez sur Module pour récupérer le fichier .py en cours d'utilisation.
  • Charger un nouveau module — sélectionnez un fichier .py et cliquez sur Charger. Le nouveau module prend effet immédiatement (sans redémarrage du serveur en mode développement ; un redémarrage gunicorn est nécessaire en production).
Le fichier module doit être un script Python valide avec un identifiant Python valide comme nom de fichier (lettres, chiffres, underscores — sans espaces ni tirets).
{# 7. Administrateur principal #}

7. Administrateur principal

L'administrateur principal accède à la page Paramètres ( dans la barre de navigation).

Connexion au serveur KoboToolBox

Renseignez l'URL du serveur (ex. https://kobo.ifrc.org) et le token API. Le token est disponible sur votre profil KoboToolBox à l'adresse <url-serveur>/token/?format=json. Cliquez sur « Tester & charger les formulaires » pour vérifier la connexion et obtenir la liste des formulaires disponibles.

Ajouter un formulaire

Après avoir chargé la liste des formulaires, sélectionnez-en un dans le menu déroulant et cliquez sur « Ajouter ce formulaire ». Vous pouvez ajuster la durée du cache (en secondes) pour contrôler la fréquence de synchronisation avec KoboToolBox.

Gérer les groupes

Créez des groupes nommés pour organiser l'accès aux formulaires. Pour chaque groupe, vous pouvez :

  • Affecter des formulaires (les membres du groupe verront uniquement ces formulaires).
  • Affecter des membres (utilisateurs actifs).
  • Désigner des administrateurs de groupe parmi les membres.

Les formulaires non affectés à aucun groupe sont visibles uniquement par l'administrateur principal.

Gérer les utilisateurs

La page « Gestion des utilisateurs » () liste les comptes en attente de validation et les comptes actifs. Vous pouvez :

  • Activer un nouveau compte après vérification de l'identité.
  • Désactiver temporairement un compte sans le supprimer.
  • Supprimer définitivement un compte.
  • Générer un lien de réinitialisation de mot de passe () — à transmettre à l'utilisateur par le canal de votre choix. Le lien est valable 3 jours et à usage unique.
Couleur principale

Depuis les Paramètres, vous pouvez personnaliser la couleur de l'interface avec le sélecteur de couleur ou en saisissant un code hexadécimal (ex. #00833D). Des raccourcis sont proposés pour les couleurs IFRC, Oxfam et ONU. La modification est instantanée pour tous les utilisateurs dès la prochaine page chargée.

Récupération d'accès

En cas de perte de mot de passe, connectez-vous au serveur via SSH et exécutez : 

python3 manage.py init_admin

Un lien de connexion à usage unique (valable 3 jours) s'affiche dans le terminal.
